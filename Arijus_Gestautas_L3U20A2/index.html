What is JavaScript?

It's a scripting language which enables you to create updating content, it allows you to control multimedia features, animated features and a lot more which allows you to make something look more advanced and unique. Whenever something interactive appears on a webpage, for example a map which you can zoom into or an image which can be rotated using the mouse, there's already a hint that JavaScript is being used.

How to incorporate JavaScript into web pages:

In a web page, most of the time if you're trying to implement JavaScript inside of it, you should usually use the <script></script> tags. This code line can go inside of a document header, it's mostly useful as it doesn't get in the way of the main content.


If at any moment you'd like the script to run at a specific stage when the page is rendered, it should most of the time be put at the correct stage of the document, most of the time inside of the document body. If the script size is small, then you should consider putting it all into one place and where it will be used as a whole. However, if the code is large, then you should consider putting it inside of the head function.

If the script is used on more than one page, or if the size is ridiculous, then consider putting it inside of it's own file and load it in the document header. This will prevent clutter of the code and you will avoid any syntax problems.

Limitations of JavaScript:

As any other programming language, JavaScript can have a few major drawbacks that will prevent a user from using it.

  • Client-Side JavaScript does not allow the reading or writing of files.

  • It does not have any multithreading or multiprocessor capabilities.

  • Browser support, server-side scripts always produces the same output, but different browsers sometimes can cause this to be an issue as they will struggle to understand the code. To prevent this, test your code on multiple browsers to prevent any mistakes.

  • Most of the code in JavaScript is usually executed on the client-side, bugs and any other issues can occur and therefore it can be exploited, because of this drawback people tend to disable JavaScript.
